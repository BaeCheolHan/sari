# Gemini Rules (강화 버전)

베이스: `.codex/rules/` (Codex 최적화 룰셋)

Gemini CLI는 룰 준수가 약한 경향이 있어 강화 버전 사용

## 차이점

| 항목 | 강화 내용 |
|------|-----------|
| 경고 블록 | 모든 핵심 정책에 `CAUTION:` 추가 |
| 필수 라벨 | `[MUST]` 라벨 강조 |
| 예시 | 2~3배 확장 |
| 위반 처리 | "작업 중단" 명시 |

## 파일 구조

- `00-core.md` - 핵심 정책 (강화판)
- `01-checklists.md` - 체크리스트 (강화판)
- `02-knowledge.md` - 지식 관리 (강화판)
- `03-gate.md` - 게이트 & 워크플로우 (강화판)
- `99-hotfix.md` - 핫픽스 정책
