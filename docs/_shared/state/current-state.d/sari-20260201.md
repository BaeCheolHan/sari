# Horadric Sari 현재 상태 (2026-02-01)

## 요약
- Codex MCP initialize가 `rootUri`/`workspaceFolders`를 전달하지 않음.
- Sari는 proxy에서 CWD 기반 `.codex-root` 탐색 및 env 주입으로 우회.
- 로그에 `Proxy startup` 및 `Initialize params (full)` 기록 확인.
- 글로벌 설치 경로: `~/.local/share/sari`.

## 최신 변경
- proxy 시작 시 `Proxy startup` 로그를 sari.log에 기록.
- daemon 시작 시 env 주입으로 workspace_root 설정.
- README에 rootUri 미전달 클라이언트 대응 안내 추가.

## 확인된 증거
- sari.log에 `Proxy startup` 및 `Initialize params (full)` 기록 존재.
- DB path가 `.../Documents/repositories/.codex/tools/sari/data/index.db`로 변경됨.

## 남은 이슈
